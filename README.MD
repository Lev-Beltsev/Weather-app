# 🌤 Weather App

Простое и красивое приложение для просмотра текущей погоды в любом городе мира.

![Weather App Demo](./design.png)

## ✨ Особенности

- 🏙 Поиск погоды по любому городу
- 📊 6 метеопоказателей: облачность, влажность, ветер, давление, UV-индекс, видимость
- 📱 Адаптивный дизайн

## 🛠 Технологии

- HTML5
- CSS3 (Flexbox, анимации)
- Vanilla JavaScript (ES6+)
- [Weatherstack API](https://weatherstack.com/) — погодный сервис
